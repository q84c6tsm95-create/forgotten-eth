Stats API Lost Forever Submit

Free, public API for checking claimable ETH in defunct smart contracts.

All endpoints are rate-limited to 30 requests per minute per IP. Responses are JSON. No authentication required.

Check Address

GET /api/check

Check if an Ethereum address has claimable ETH in any tracked contract.

ParameterTypeDescription
address requiredstringEthereum address (0x + 40 hex chars)

Example

GET /api/check?address=0x1234567890abcdef1234567890abcdef12345678

Response

{
  "address": "0x1234...5678",
  "checked_at": "2026-03-31T12:00:00.000Z",
  "contracts_checked": 116,
  "contracts_with_balance": 2,
  "total_claimable_eth": "1.234567",
  "balances": {
    "etherdelta": {
      "contract": "0x8d12A197cB00D4747a1fe03395095ce2A5CC6819",
      "balance_wei": "1234567000000000000",
      "balance_eth": "1.234567"
    }
  },
  "coverage": {
    "etherdelta": { "coverage_pct": 99.8, "scan_date": "2026-03-30 UTC" }
  }
}

Protocol Table

GET /api/table

Get metadata for a specific protocol: total ETH, address count, coverage, TVL history.

ParameterTypeDescription
exchange requiredstringProtocol key (e.g. etherdelta, ens_old)

Example

GET /api/table?exchange=etherdelta

Response

{
  "exchange": "etherdelta",
  "meta": {
    "contract": "0x8d12A197cB00D4747a1fe03395095ce2A5CC6819",
    "total_eth": 1234.56,
    "contract_eth_balance": 1500.00,
    "addresses_with_balance": 45000,
    "coverage_pct": 99.8,
    "scan_date": "2026-03-30 UTC",
    "tvl": [
      { "date": "2018-01", "eth": 150000 },
      { "date": "2026-03", "eth": 1500 }
    ]
  }
}

Site Stats

GET /api/stats

Aggregate usage stats: how many addresses checked, balances found, ETH claimed.

Example

GET /api/stats

Response

{
  "total_checks": 5000,
  "unique_checked": 3200,
  "total_found": 800,
  "unique_found": 650,
  "total_claims": 42,
  "eth_claimed": 125.50
}

Protocol List

GET /data/protocols.json

Static JSON file listing all tracked protocols with their keys, names, and categories.

Example

GET /data/protocols.json

Response

[
  {
    "key": "etherdelta",
    "name": "EtherDelta",
    "category": "DEX",
    "balance_file": "etherdelta.json"
  },
  ...
]

Notes

  • Rate limit: 30 requests/min per IP. Returns 429 when exceeded.
  • All ETH values are in ether (not wei), unless the field name says _wei.
  • Coverage percentage shows what fraction of known depositors are in our index.
  • Data refreshes every 6 hours. The scan_date field shows when each contract was last scanned.
  • Individual address balances are only returned via /api/check. The /api/table and /api/summary endpoints return aggregate data only.
